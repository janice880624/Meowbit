# Meowbit

### 產品規格

* MCU：STM32F401RET6, 32-bit ARM Cortex M4 core
* Operating voltage: 3.3V
* Supply voltage：USB\(5V\), Lithium battery pack\(3.7~4.2V\)
* Output current: 500mA\(max\)
* Dimensions: 52x76x12\(mm\)
